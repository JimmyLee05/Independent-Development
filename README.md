# FirstMet App运营的经历

开发一个App后，运营工作的好坏能直接决定一款App的成败。毕竟一款App做的再好，用户没有机会知道也是不行的。

这是一个流量最贵的时代。

这段经历让我入门了运营工作，认识了很多运营大佬，也知道了运营工作的流程。

对自己的App推广有很大帮助，能从更多的角度看待一款App。
